### Usage
run docker-compose.yml file to run mysql and phpmyadmin containers
```
docker-compose up -d --build
```
Run front
```
cd front
npm i
npm start
```

Run CRUD http server
```
cd server
npm i
npm run dev
```

Run socker server
```
cd socket-server (Not available, on process)
npm i
```


http://localhost:3000
